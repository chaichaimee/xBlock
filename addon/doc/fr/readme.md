## **Le moteur de productivité ultime**

**xBlock** est une extension NVDA professionnelle conçue pour l'efficacité. Elle transforme la gestion des textes répétitifs en créant une bibliothèque structurée de fragments. Gagnez du temps avec vos codes, emails types ou formulaires.

<br>

## **Commandes clés**

> [!TIP]
> **Windows + F11 :** Ouvrir le dialogue xBlock.
>
> *Pour modifier ce raccourci, allez dans le **Menu NVDA > Préférences > Gestes de commande**, et cherchez "xBlock" dans Global Plugins.*

<br>

## **Comment ça marche : Étape par étape**

1. **Lancer :** Appuyez sur **Windows + F11** pour ouvrir xBlock.
2. **Organiser :** Choisissez la catégorie **"Tous"** ou créez une **"Nouvelle catégorie"**.
3. **Créer :** Entrez le **Nom du bloc** et son **Contenu**.
4. **Enregistrer :** Cliquez sur **"Ajouter"**.
5. **Coller :** Sélectionnez un bloc et appuyez sur **Enter** pour coller instantanément.
