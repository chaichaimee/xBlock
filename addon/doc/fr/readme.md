<h2 align="center"><b>L'ultime moteur de productivité</b></h2>

**xBlock** est une extension NVDA de qualité professionnelle conçue pour les utilisateurs qui privilégient l'efficacité. Il transforme votre façon de gérer les textes répétitifs en créant une bibliothèque structurée de fragments. Qu'il s'agisse de code complexe, de réponses types par e-mail ou de formulaires fréquemment utilisés, xBlock garantit que votre contenu est organisé et prêt à être déployé à la vitesse de la pensée.

<h2 align="center"><b>Commandes principales</b></h2>

> **Windows + F11 :** Ouvrir la boîte de dialogue xBlock.
>
> _*Pour modifier ce raccourci, allez dans le **Menu NVDA > Préférences > Gestes de commande**, et recherchez "xBlock" sous la catégorie Plugins globaux._

<h2 align="center"><b>Comment ça marche : Étape par étape</b></h2>

1.  **Lancement :** Appuyez sur **Windows + F11** pour ouvrir le centre de commande xBlock.
2.  **Organisation :** Choisissez la catégorie **"Tout"** ou cliquez sur **"Nouvelle catégorie"** pour créer un groupe personnalisé pour vos blocs.
3.  **Création :** Entrez un **Nom de bloc** (le titre que vous verrez dans la liste) et tapez ou collez votre texte dans le champ **Contenu du bloc**.
4.  **Sauvegarde :** Cliquez sur le bouton **"Ajouter"** pour stocker votre nouveau fragment.
5.  **Collage :** Sélectionnez simplement votre bloc dans la liste et appuyez sur **Entrée**. La fenêtre se fermera et votre texte sera instantanément collé dans votre application active.

<h2 align="center"><b>Capacités avancées</b></h2>

* **Catégorisation intelligente :** Gérez sans effort des centaines de blocs en les regroupant dans des catégories dédiées.
* **Gestion des priorités :** Faites un clic droit sur n'importe quel bloc pour l'**Épingler** en haut. Utilisez **Déplacer vers le haut/bas** pour organiser vos fragments les plus importants exactement là où vous le souhaitez.
* **Édition fluide :** Faites un clic droit pour modifier le contenu, déplacer des blocs vers d'autres catégories ou supprimer instantanément des éléments indésirables.
