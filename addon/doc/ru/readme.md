## **Максимальная производительность**

**xBlock** — это профессиональное дополнение для NVDA, созданное для тех, кто ценит время. Оно превращает работу с повторяющимся текстом в удобный процесс с помощью структурированной библиотеки фрагментов.

<br>

## **Основные команды**

> [!TIP]
> **Windows + F11:** Открыть окно xBlock.
>
> *Изменить эту комбинацию можно в **Меню NVDA > Параметры > Жесты ввода**, раздел Global Plugins (xBlock).*

<br>

## **Как это работает: шаг за шагом**

1. **Запуск:** Нажмите **Windows + F11**.
2. **Организация:** Выберите категорию **"Все"** или создайте **"Новую категорию"**.
3. **Создание:** Введите **Имя блока** и его **Содержимое**.
4. **Сохранение:** Нажмите кнопку **"Добавить"**.
5. **Вставка:** Выберите блок из списка и нажмите **Enter** для мгновенной вставки.
