<h2 align="center"><b>Das ultimative Produktivitäts-Kraftpaket</b></h2>

**xBlock** ist ein professionelles NVDA-Add-on für Benutzer, die Wert auf Effizienz legen. Es verändert den Umgang mit sich wiederholenden Texten, indem es eine strukturierte Bibliothek von Snippets erstellt. Ob komplexer Code, Standard-E-Mail-Antworten oder häufig verwendete Formulare – xBlock sorgt dafür, dass Ihre Inhalte organisiert und blitzschnell einsatzbereit sind.

<h2 align="center"><b>Wichtige Befehle</b></h2>

> **Windows + F11:** Öffnet den xBlock-Dialog.
>
> _*Um diesen Shortcut zu ändern, gehen Sie zum **NVDA-Menü > Einstellungen > Eingabegesten**, und suchen Sie nach „xBlock“ unter der Kategorie Global Plugins._

<h2 align="center"><b>So funktioniert es: Schritt für Schritt</b></h2>

1.  **Starten:** Drücken Sie **Windows + F11**, um das xBlock Command Center zu öffnen.
2.  **Organisieren:** Wählen Sie die Kategorie **"Alle"** oder klicken Sie auf **"Neue Kategorie"**, um eine benutzerdefinierte Gruppe für Ihre Blöcke zu erstellen.
3.  **Erstellen:** Geben Sie einen **Blocknamen** ein (der Titel in der Liste) und tippen oder fügen Sie Ihren Text in das Feld **Blockinhalt** ein.
4.  **Speichern:** Klicken Sie auf die Schaltfläche **"Hinzufügen"**, um Ihr neues Snippet zu speichern.
5.  **Einfügen:** Wählen Sie einfach Ihren Block aus der Liste aus und drücken Sie **Enter**. Das Fenster schließt sich und Ihr Text wird sofort in Ihre aktive Anwendung eingefügt.

<h2 align="center"><b>Erweiterte Funktionen</b></h2>

* **Intelligente Kategorisierung:** Verwalten Sie mühelos hunderte von Blöcken, indem Sie sie in speziellen Kategorien gruppieren.
* **Prioritätsmanagement:** Klicken Sie mit der rechten Maustaste auf einen Block, um ihn **oben anzuheften**. Verwenden Sie **Nach oben/unten verschieben**, um Ihre wichtigsten Snippets genau dort zu platzieren, wo Sie sie benötigen.
* **Nahtlose Bearbeitung:** Rechtsklick zum Bearbeiten von Inhalten, Verschieben von Blöcken in andere Kategorien oder sofortiges Löschen unerwünschter Elemente.
