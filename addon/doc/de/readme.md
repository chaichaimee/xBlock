## **Das ultimative Produktivitäts-Kraftpaket**

**xBlock** ist ein professionelles NVDA-Add-on für Benutzer, die Wert auf Effizienz legen. Es verändert den Umgang mit sich wiederholenden Texten durch eine strukturierte Snippet-Bibliothek. Ob komplexer Code, Standard-E-Mail-Antworten oder häufig genutzte Formulare – xBlock stellt sicher, dass Ihre Inhalte organisiert und sofort einsatzbereit sind.

<br>

## **Wichtige Befehle**

> [!TIP]
> **Windows + F11:** Öffnet den xBlock-Dialog.
>
> *Um diesen Shortcut zu ändern, gehen Sie zu **NVDA-Menü > Einstellungen > Eingaben** und suchen Sie nach "xBlock" unter der Kategorie Global Plugins.*

<br>

## **So funktioniert es: Schritt für Schritt**

1. **Starten:** Drücken Sie **Windows + F11**, um das xBlock-Center zu öffnen.
2. **Organisieren:** Wählen Sie die Kategorie **"Alle"** oder erstellen Sie eine **"Neue Kategorie"**.
3. **Erstellen:** Geben Sie einen **Blocknamen** und den **Inhalt** ein.
4. **Speichern:** Klicken Sie auf **"Hinzufügen"**.
5. **Einfügen:** Wählen Sie den Block aus und drücken Sie **Enter** – der Text wird sofort eingefügt.

<br>

## **Erweiterte Funktionen**

* **Intelligente Kategorisierung:** Verwalten Sie hunderte Blöcke mühelos in eigenen Kategorien.
* **Prioritäts-Management:** Rechtsklick zum **Anheften** oben; nutzen Sie **Nach oben/unten verschieben** zur Sortierung.
* **Nahtlose Bearbeitung:** Bearbeiten, Verschieben oder Löschen von Elementen über das Kontextmenü.
