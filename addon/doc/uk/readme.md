## **Максимальна продуктивність**

**xBlock** — це професійне доповнення для NVDA для тих, хто цінує ефективність. Воно впорядковує роботу з повторюваним текстом завдяки структурованій бібліотеці фрагментів.

<br>

## **Основні команди**

> [!TIP]
> **Windows + F11:** Відкрити вікно xBlock.
>
> *Змінити цю комбінацію можна в **Меню NVDA > Параметри > Жести вводу**, розділ Global Plugins (xBlock).*

<br>

## **Як це працює: крок за кроком**

1. **Запуск:** Натисніть **Windows + F11**.
2. **Організація:** Виберіть категорію **"Усі"** або створіть **"Нову категорію"**.
3. **Створення:** Введіть **Назву блоку** та його **Вміст**.
4. **Збереження:** Натисніть кнопку **"Додати"**.
5. **Вставка:** Виберіть блок зі списку та натисніть **Enter** для миттєвої вставки.
